# Pathfinder

Pathfinder is a full-stack application designed to help users find and navigate optimal routes. The project is organized with clear separation between backend and frontend components.

## Project Structure

```
pathfinder/
├── src/
│   ├── backend/     # Backend API and server logic
│   └── frontend/    # Frontend web application
└── ...
```

### Backend (`src/backend`)

The backend component contains the server-side logic, APIs, and business logic for the Pathfinder application.

### Frontend (`src/frontend`)

The frontend component contains the web application user interface and client-side logic.

## Getting Started

1. Navigate to the backend directory and follow the setup instructions
2. Navigate to the frontend directory and follow the setup instructions
3. Configure any necessary environment variables
4. Start the development servers
